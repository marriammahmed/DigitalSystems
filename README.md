<h1>Digital Systems: Logic Design, Latches & Sequential Circuits</h1>

<h2>Description</h2>
This repository contains my work from the <b>Digital Systems</b> course, focused on building hardware from the ground up using logic design principles.  
The course covered <b>combinational and sequential circuit design</b>, including latches, flip-flops, registers, counters, and finite state machines.

<br/>

<b>Course Outcomes & Skills:</b><br/>
- Working with <b>number systems</b> (binary, octal, hexadecimal) and conversions<br/>
- Applying <b>Boolean algebra</b> and logic minimization (Karnaugh Maps)<br/>
- Designing <b>combinational circuits</b> (adders, multiplexers, decoders, encoders, ALU basics)<br/>
- Understanding and implementing <b>SR, D, JK, and T latches & flip-flops</b><br/>
- Designing <b>registers</b> (shift registers, parallel registers)<br/>
- Building <b>counters</b> (asynchronous & synchronous counters)<br/>
- Designing <b>finite state machines (FSM)</b><br/>
- Understanding <b>clocking, propagation delay, setup/hold time</b><br/>
- Simulating and testing digital circuits using hardware design tools<br/>
- Strengthening hardware-level understanding essential for computer architecture and low-level systems<br/>

<br/>

<h2>Languages & Tools Used</h2>
- <b>Digital Logic Design</b><br/>
- <b>Boolean Algebra</b><br/>
- <b>Logisim</b><br/>
- <b>Hardware Simulation Tools</b><br/>
- <b>Xcode / VS Code</b><br/>
